# NKOD graph browser

Simple web-based graph browser for ontologies used by NKOD datasets.

## Build instructions

- Optionally create `./.env.local` file that overrides `./.env`
- Run `npm install` to install all dependencies
- Run `npm run build` to build the project
- Publish the `./build` directory
